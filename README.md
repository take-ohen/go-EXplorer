go-EXplorer

Break the conventional file browsing. Speed, Insight, and Extra.
🚀 Concept

既存の「フォルダを辿る」「キーワードを打つ」だけのファイル管理を打破し、Goの並行処理（Goroutine）を活かした爆速なスキャンと、情報の「密度」を視覚化する新しいブラウジング体験を提供します。
🛠 First Goals

    [ ] Extreme Scan: Goroutineによる高速なディレクトリ走査

    [ ] EX-grep: 全域を瞬時に突き抜ける検索機能

    [ ] Visual Sum: ファイルサイズを「塊」として捉えるグラフ表示（FileSum的アプローチ）
